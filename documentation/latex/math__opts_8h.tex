Header containing number generators. The header file contains the random number generator and primerizing functions. The random numbers generators work with blum blum shub and G\-M\-P library functions.

The function implemented with G\-M\-P functions for working easily with big numbers. It does modulo operation to exponential functions quickly. Since the for loop inside the function loops for the bit length of the exponential number, the function runs very fast. For example if we have 1024 bit key and we are doing the encryption, the loop only loops for 1024 times. It turns out O(n) is the run time for the whole encryption.
