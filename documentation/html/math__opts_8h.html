Detailed Description

Header containing number generators.

The header file contains the random number generator and primerizing functions. The random numbers generators work with blum blum shub and GMP library functions.

Does exponentiation for the encryption process to the very large numbers.

Parameters:
fThe remaining.
baseBase number
powerPower of the number
modModulo number

The function implemented with GMP functions for working easily with big numbers. It does modulo operation to exponential functions quickly. Since the for loop inside the function loops for the bit length of the exponential number, the function runs very fast. For example if we have 1024 bit key and we are doing the encryption, the loop only loops for 1024 times. It turns out O(n) is the run time for the whole encryption.
